# Tiny Hamburger

Part of Tiny Components, small Helpers for Frontend Development. This is a Hamburger Menu.
It used the Framework [RiotJS v4](https://riot.js.org/).

## Usage

```html
<tiny-hamburger>
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">Blog</a></li>
            <li><a href="#">Contact</a></li>
        </ul>
    </nav>
</tiny-hamburger>
```
